# Dataset Description

This project analyzes a structured higher education dataset containing university and tuition-related information.

The dataset includes:
- Basic university attributes (name, region, establishment type)
- Tuition records segmented by year, department, and academic structure

The data was used to explore institutional differences in tuition levels and regional distribution patterns through SQL-based analysis.

Analytical workflow includes:
- Data filtering to define the analysis scope
- Aggregation to understand regional distribution
- JOIN operations to integrate relational tables
- CTE-based transformations for multi-step analysis
- Pivot-style restructuring for time-series comparison

Note:
The original dataset is not included due to licensing restrictions.

# University Tuition SQL Analysis

## Overview

This mini project explores educational institution and tuition datasets using SQL.

The analysis focuses on practicing:
- filtering
- aggregation
- joins
- CTEs
- pivot-style reporting

The goal was to build foundational SQL workflows for transforming relational data into structured analytical outputs.

---

## Key Questions

- Which regions contain the highest number of institutions?
- How can tuition data be integrated with school metadata?
- How do average tuition values vary across years and institutions?
- How can SQL transform row-level data into reporting-friendly summaries?

---

## Skills Practiced

- SELECT / WHERE
- GROUP BY / ORDER BY
- LEFT JOIN
- Common Table Expressions (CTEs)
- CASE WHEN pivoting

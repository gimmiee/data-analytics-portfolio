# Key Findings

## 1. Tuition levels vary significantly across universities
Analysis showed that tuition fees differ noticeably depending on the institution, with some universities consistently positioned in higher fee ranges compared to others.

## 2. Universities are concentrated in specific regions
The regional distribution is not balanced, with a higher number of institutions clustered in certain areas rather than being evenly spread.

## 3. Tuition trends show divergence over time
While some universities show stable tuition patterns across years, others demonstrate noticeable increases, indicating inconsistent pricing strategies.

## 4. SQL enabled multi-step analytical transformation
Using JOINs, aggregation, and CTE-based structuring allowed raw relational data to be transformed into analysis-ready datasets for trend interpretation.
